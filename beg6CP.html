After 6CO, the 6 corners need to be permuted with respect to each other. In the beginner variation, we do this in 2 steps. First, we locate the DBR corner, and if it is not already in DBR, we place it there using R2 U R2 (if it is in DFR) or R2 U' R2 (if it is in UFL). This leaves us with permutation of the last 5 corners, which we solve algorithmically.

If the DFR corner also happens to be solved, we get the familiar Y perm or J perm cases. Otherwise, the DFR corner is to be placed in UBL, and the case is recognised by the corner stickers on the F and R faces in the top layer; there are 6 cases in this catagory. Note that the algorithms offset the D layer with a D', so that the last 5 edges can be solved using MU-gen algorithms without having to rotate.
